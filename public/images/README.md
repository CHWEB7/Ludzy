# Brand assets

- **`ludzy-logo.png`** — Export the tubular **LUDZY** wordmark (transparent or black backing) here. Header uses fixed height with `object-contain` (~44px tall on desktop).

- **`hero-light-trails.jpg`** — Optional hero background for the wide “Introducing” panel. Set `heroMainTextureSrc` in `app/page.tsx` to `"/images/hero-light-trails.jpg"` when you drop the file here (long‑exposure light trails / night energy works best).

Optional:

- **`evening-social.jpg`** — Replace the first hero thumbnail by changing `heroPhotos[0].src` in `app/page.tsx` to `"/images/evening-social.jpg"`.
